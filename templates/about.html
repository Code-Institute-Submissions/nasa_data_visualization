About

    I created this website as a study of Extra-vehicular Activities using all open source tools. The data in the dataset used on this site was last updated on April 7, 2015, so any EVAs after that date aren't included on this site unfortunately.

    All links have been provided to the source material of the relevant data, videos and images. You can check out the code for this site on my Github here.
